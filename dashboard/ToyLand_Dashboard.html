// The Journey

From data to autonomy.

Four stages. Each uses our BizBotBash backend plus Claude Cowork to build the layer above.

Run a sim, download the excel files, hand them to Claude, ask for insights.

  1. Run a simulation — pick a bot, 60 months, click Run. Calls our backend.
  2. Download the excel bundle — zip of initial_state.xlsx + month_*.xlsx
  3. Create ~/toyland-data/, unzip into it
  4. Install Claude Cowork, point it at the folder
  5. Drop in toyland-analysis.md and tell Claude to execute it
  6. Claude builds aggregate.py, analysis.py, charts.py, insights.md
> .ZIP Excel bundle > .MD Cowork prompt
// Outcome
A local data folder, Python scripts that aggregate + analyze, and a concrete sense of how the business behaves.

Extend the Python from Stage 01 into a dashboard app that writes back to the excel files.

  1. Ask Claude to add write functions to aggregate.py (issue PO, record transfer, update shelf)
  2. Build a small Flask UI with pages for each action
  3. Wire "Advance one month" to call our backend at /run or /run-bot
  4. Style it like BizBotBash — Claude will match the theme for you
// Outcome
A working ERP. Click buttons, excel files update, business moves forward.

Plug in a bot that recommends the next moves. Human approves or ignores.

  1. Fork bots/bizbotbash_champion.py, return actions as suggestions + rationale
  2. Add a Recommendations panel to your ERP with Accept / Ignore buttons
  3. Accept → reuse Stage 02 write functions. Ignore → log the disagreement.
  4. After a month, ask Claude: "Which actions am I accepting >80% of the time?"
// Outcome
AI assistant inside your ERP. Trust, built step by step, with a human safety net.
